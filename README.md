# Employee Performance Management System (PL/SQL)

###  Overview
This project demonstrates the use of **PL/SQL Collections**, **Records**, and **GOTO statements** through a real-world example of employee performance analysis.

###  Objectives
- Manage structured employee data using Records.
- Store and process multiple employees with Collections.
- Handle invalid data entries using GOTO control flow.
- Display top and low performers dynamically.

###  How to Run
1. Open Oracle SQL Developer or any PL/SQL environment.
2. Run `scripts/employee_performance.sql`.
3. View the console output to see processed results.

###  Structure
- `scripts/`: Contains main PL/SQL code and optional data script.
- `docs/`: Project explanation and design overview.
- `output/`: Execution results and screenshots.
